## Dependencies

[yt-dlp](https://github.com/yt-dlp/yt-dlp)
[mpv](https://mpv.io/)

## Installation

<details>
  <summary>Cargo</summary>

  ```bash
  cargo install ani-fr
  ```
</details>

## Build
```bash
git clone https://github.com/merlinloveboobs/ani-fr.git
cd ani-fr
cargo build --release
```

## Usage

After installation, run:
```bash
ani-fr
```
